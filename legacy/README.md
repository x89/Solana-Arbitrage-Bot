# Archived prototypes

This directory contains historical experiments preserved for source archaeology.
Nothing here is part of the supported build, CI, release, or deployment surface.

The archived crates contain obsolete Solana/Anchor dependencies and protocol
interfaces. Do not fund, deploy, reactivate, or modernize them through blanket
dependency upgrades. Any future execution component requires a clean,
current-protocol implementation and an independent security review.

See [`../MIGRATION.md`](../MIGRATION.md) for the dated technical findings.
